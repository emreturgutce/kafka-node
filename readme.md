### Starting Kafka and Zookeeper

```bash
docker-compose up
```

### Creating Topic

```bash
yarn create:topic <TOPIC>
```

### Send Message

```bash
yarn send:message <MESSAGE>
```

### Subscribe To a Topic

```bash
yarn subscribe:topic <TOPIC>
```
